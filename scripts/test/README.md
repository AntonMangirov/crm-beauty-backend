# Тестовые скрипты

Эта папка содержит PowerShell скрипты для ручного тестирования API.

## Использование

Все скрипты находятся в этой папке. Для запуска:

```powershell
cd scripts\test
.\test-api.ps1
.\test-booking-simple.ps1
# и т.д.
```

## Файлы

- `test-*.ps1` - PowerShell скрипты для тестирования различных эндпоинтов
- `fixtures/` - JSON файлы с тестовыми данными

## Примечание

Эти скрипты используются для ручного тестирования. Для автоматических тестов используйте Jest тесты в `src/__tests__/`.
